How PhishGuard Works

Our advanced hybrid model combines two powerful machine learning techniques

LSTM Neural Network

Long Short-Term Memory (LSTM) networks analyze URL patterns at the character level, capturing sequential information and understanding the structure of phishing URLs.

  • Character pattern recognition
  • Sequential analysis
  • Deep learning capabilities

Random Forest Classifier

Random Forest analyzes extracted features from URLs to identify patterns common in phishing sites, using multiple decision trees to make robust predictions.

  • Feature-based analysis
  • High interpretability
  • Resistant to overfitting
